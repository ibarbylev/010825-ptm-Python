## –°–ª–æ–≤–∞—Ä—å `dict`

‚Äî —ç—Ç–æ –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–∞—è (–¥–æ Python3.7!) –∏–∑–º–µ–Ω—è–µ–º–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è –ø–∞—Ä **–∫–ª—é—á‚Äì–∑–Ω–∞—á–µ–Ω–∏–µ**,
–≥–¥–µ –∫–∞–∂–¥–æ–º—É **—É–Ω–∏–∫–∞–ª—å–Ω–æ–º—É** `hasible` –∫–ª—é—á—É —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç **–æ–¥–Ω–æ** –∑–Ω–∞—á–µ–Ω–∏–µ.

–í –æ—Å–Ω–æ–≤–µ —Å–ª–æ–≤–∞—Ä—è –ª–µ–∂–∏—Ç —Ö—ç—à-—Ç–∞–±–ª–∏—Ü–∞.

```python
person = {"name": "Alice", "age": 25}
```

* `"name"`, `"age"` ‚Äî –∫–ª—é—á–∏
* `"Alice"`, `25` ‚Äî –∑–Ω–∞—á–µ–Ω–∏—è

## –ü–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ –∫–ª—é—á—É

–î–æ—Å—Ç—É–ø –∫ –∑–Ω–∞—á–µ–Ω–∏—è–º –≤–æ–∑–º–æ–∂–µ–Ω –ø–æ –∫–ª—é—á–∞–º:

```python
person = {"name": "Alice", "age": 25}

print(person["name"])  # Alice
```

–ü–æ–ø—ã—Ç–∫–∞ –ø–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ –Ω–µ—Å—É—â–µ—Å—Ç–≤—É—é—â–µ–º—É –∫–ª—é—á—É –¥–∞—ë—Ç –æ—à–∏–±–∫—É:

```python
person = {"name": "Alice", "age": 25}

print(person["surname"])  # KeyError: 'surname'
```

–ü–æ—ç—Ç–æ–º—É, —É–¥–æ–±–Ω–µ–µ –ø–æ–ª—É—á–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ —Å –ø–æ–º–æ—â—å—é –º–µ—Ç–æ–¥–∞ `.get(key[, default])`:
* –ï—Å–ª–∏ –∫–ª—é—á –µ—Å—Ç—å, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ
* –ï—Å–ª–∏ –Ω–µ—Ç, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ –¥–µ—Ñ–æ–ª—Ç—É (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é None) 

```python
person = {"name": "Alice", "age": 25}

print(person.get("name", "Bob"))  # "Alice"
print(person.get("surname"))  # None
```

## –î–æ–±–∞–≤–∏—Ç—å / –∏–∑–º–µ–Ω–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ –∫–ª—é—á—É

```python
person = {"name": "Alice", "age": 25}
person["city"] = "Paris"

```

## –ú–æ–∂–Ω–æ –ª–∏ –≤ —Å–ª–æ–≤–∞—Ä–µ –±—ã—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –æ–¥–∏–Ω–∞–∫–æ–≤—ã—Ö –∫–ª—é—á–µ–π?

–ï—Å–ª–∏ —É –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –ø–∞—Ä –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π –∫–ª—é—á (—Ç–æ—á–Ω–µ–µ, –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π —Ö—ç—à!), —Ç–æ –æ—Å—Ç–∞–Ω–µ—Ç—Å—è –ø–æ—Å–ª–µ–¥–Ω—è—è –ø–æ—Ä–∞:

## 4. dict comprehension

# –ö–≤–∞–¥—Ä–∞—Ç—ã —á–∏—Å–µ–ª

## 5. –í–∞–∂–Ω–æ–µ –ø—Ä–∞–≤–∏–ª–æ —Å–ª–æ–≤–∞—Ä—è: "–∫–ª—é—á - –≤—Å—ë, –∑–Ω–∞—á–µ–Ω–∏–µ - –Ω–∏—á—Ç–æ"

–í —Å–ª–æ–≤–∞—Ä–µ –≥–ª–∞–≤–Ω—ã–º —ç–ª–µ–º–µ–Ω—Ç–æ–º —è–≤–ª—è–µ—Ç—Å—è **–∫–ª—é—á**, –ø–æ—ç—Ç–æ–º—É –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –∏ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–π Python  
–ø—Ä–∏–º–µ–Ω—è—é—Ç—Å—è **—Ç–æ–ª—å–∫–æ –∫ –∫–ª—é—á–∞–º**, –∞ –Ω–µ –∫ –∑–Ω–∞—á–µ–Ω–∏—è–º.

### 5.1. –û–ø–µ—Ä–∞—Ç–æ—Ä `in` / `not in`

–ü—Ä–æ–≤–µ—Ä—è–µ—Ç **–Ω–∞–ª–∏—á–∏–µ –∫–ª—é—á–∞**, –∞ –Ω–µ –∑–Ω–∞—á–µ–Ω–∏—è.

```python
d = {"a": 1, "b": 2}

print("a" in d)     # ‚úÖ True ‚Äî –∫–ª—é—á —Å—É—â–µ—Å—Ç–≤—É–µ—Ç
print(1 in d)       # ‚ùå False ‚Äî 1 —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ, –Ω–µ –∫–ª—é—á
```

üëâ –ö—Å—Ç–∞—Ç–∏, –ø–æ–∏—Å–∫ –≤ –∑–Ω–∞—á–µ–Ω–∏—è—Ö —Ç–æ–∂–µ –≤–æ–∑–º–æ–∂–µ–Ω:

```python
1 in d.values()     # ‚úÖ True
```

---

### 5.2. –û–ø–µ—Ä–∞—Ç–æ—Ä `del`

–£–¥–∞–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç **–ø–æ –∫–ª—é—á—É**.

```python
del d["a"]      # —É–¥–∞–ª–∏—Ç –ø–∞—Ä—É "a": 1
# del 1 ‚ùå ‚Äî –Ω–µ–ª—å–∑—è —É–¥–∞–ª–∏—Ç—å –ø–æ –∑–Ω–∞—á–µ–Ω–∏—é
```

---

### 5.3. –¶–∏–∫–ª `for`

–ü—Ä–∏ –∏—Ç–µ—Ä–∞—Ü–∏–∏ –ø–æ —Å–ª–æ–≤–∞—Ä—é –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç—Å—è **–∫–ª—é—á–∏**:

```python
for key in d:
    print(key)
# –í—ã–≤–µ–¥–µ—Ç: a, b
```

–î–ª—è –∏—Ç–µ—Ä–∞—Ü–∏–∏ –ø–æ –∑–Ω–∞—á–µ–Ω–∏—è–º –∏–ª–∏ –ø–∞—Ä–∞–º —É–¥–æ–±–Ω—ã –º–µ—Ç–æ–¥—ã `.values()` –∏ `.items()`:

```python
for value in d.values():
    print(value)

for key, value in d.items():
    print(key, value)
```

### 5.5. –ò—Ç–æ–≥–∏ –ø—Ä–∞–≤–∏–ª–∞ "–∫–ª—é—á - –≤—Å—ë, –∑–Ω–∞—á–µ–Ω–∏–µ - –Ω–∏—á—Ç–æ"
